| CH | Function       | Range   | Notes                                              |
|----|---------------|---------|-----------------------------------------------------|
| 1  | Master Dimmer | 0-255   | Total dimming (used in combination with CH5-7)      |
| 2  | Red           | 0-255   |                                                     |
| 3  | Green         | 0-255   |                                                     |
| 4  | Blue          | 0-255   |                                                     |
| 5  | White         | 0-255   |                                                     |
| 6  | Strobe        | 0-255   | Higher value = faster speed                         |
| 7  | Mode Select   | 0-255   | Color selection / effects                           |
